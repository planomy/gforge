# Critter eggs & sidekicks

One critter per **completed theme** (6/6 stories). When a theme is finished, the
**next arcade run** opens with the **birth ship** cinematic, then a reveal card.
The egg sits in the **topbar nest** and hatches in **2–5 days** (random). After
hatch, the critter is a **cosmetic arcade sidekick** (no combat power).

Drop your art here (PNG, transparent where possible):

```
pets/
  sport/
    birth_ship.png   — ship that delivers the egg (pregame cinematic)
    card.png         — cute reveal card shown after delivery
    egg.png          — egg in the topbar nest
    critter.png      — hatched sidekick (arcade + nest)
  adventure/  …same four files…
  mystery/    …
  scifi/      …
  fantasy/    …
  horror/     …
```

Until files exist, the game shows topic-coloured placeholder eggs/critters.

### Baby gallery cards (`cards/`)

Transparent trading-card art per theme (see `cards/README.md`). Used for the
home-screen baby card carousel.
