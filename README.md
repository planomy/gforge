# gforce

Forge powerful English sentences — a guided writing tool for 8–12 year-old native speakers who need help moving past basic sentences into sharper, more vivid prose.

## How it works

- Pick a topic (**Sport, Adventure, Mystery, Sci-Fi**) and a story.
- Each story is a **5-sentence narrative paragraph**.
- For each sentence, you're shown a fixed opener and a few containers of word/phrase choices.
- Pick the **best option** in each container — distractors are designed to teach against vague nouns, basic verbs, filter words and tell-not-show writing.
- Press **Check**. Wrong picks turn red so you can fix them.
- Once the sentence is correct, **type it out** word-for-word as reinforcement.
- After all 5 sentences, your **full paragraph** is revealed and saved to **My Stories**.
- Finishing a story unlocks a **60-second arcade reward game**.

## Stories

24 stories total — 6 per topic, split across 3 difficulty levels (L1, L2, L3).

## Run locally

Just open `index.html` in a browser. All state is saved in `localStorage`.
